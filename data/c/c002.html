步骤：
1. 写代码
## 先安装vim
$sudo apt-get install  vim
## 编辑自己的hello word
$vim  hello.c
## 输入小写英文 "i" 就可以进行编辑，输入如下内容
#include <stdio.h>
int main(){
     printf("hello,world\n");
     return  0;
}
## 编写完成后 按esc键 然后输入 ":wq" 然后enter进行保存


2. gcc 编译
## 编译后会多出一个编译好的可执行程序文件
## gcc编译C语言的最简命令为：gcc c_files -o target_name

$ gcc hello.c -o hello


3 运行你编译好的c程序
## 例如我这里是hello
$ ./hello
hello,world	




或者使用 make 编译

float 的二进制表示

float 是8字节，s/exp/frac字段分别是1位, k=8位和n=32位，得到一个32位的表示。31, [30,23], [22,0]

double 是8字节，s/exp/frac字段分别是1位, k=11位和n=52位，得到一个64位的表示。63, [62,52], [51,0]
