2. 完整代码: 带自增行号的web编辑器，通过给textarea添加行号

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>📋 Line Numbers for Textarea</title>
    <style>
      .editor {
        display: inline-flex;
        gap: 10px;
        font-family: monospace;
        line-height: 21px;
        background: #282a3a;
        border-radius: 2px;
        padding: 20px 10px;
        height: 200px;
        overflow-y: auto;
      }

      .line-numbers {
        width: 20px;
        text-align: right;
        height: 9999px;
      }

      .line-numbers span {
        counter-increment:  linenumber;
      }

      .line-numbers span::before {
        content: counter(linenumber);
        display: block;
        color: #506882;
      }

      textarea {
        height: 9999px;
        line-height: 21px;
        overflow-y: hidden;
        padding: 0;
        border: 0;
        background: #282a3a;
        color: #FFF;
        min-width: 500px;
        outline: none;
        resize: none;
      }
    </style>
  </head>
  <body>
    <div class="editor">
      <div class="line-numbers">
        <span></span>
      </div>
      <textarea></textarea>
    </div>

    <script>
      const textarea = document.querySelector('textarea')
      const lineNumbers = document.querySelector('.line-numbers')

      textarea.addEventListener('keyup', event => {
        const numberOfLines = event.target.value.split('\n').length

        lineNumbers.innerHTML = Array(numberOfLines)
          .fill('<span></span>')
          .join('')
      })

      textarea.addEventListener('keydown', event => {
        if (event.key === 'Tab') {
          const start = textarea.selectionStart
          const end = textarea.selectionEnd

          textarea.value = textarea.value.substring(0, start) + '\t' + textarea.value.substring(end)
          textarea.focus()      

          event.preventDefault()
        }
      })
    </script>
  </body>
</html>

一行几个连续的方框，展示颜色效果：怎么避免间隔？ | html几个div组成的颜色块，互相之间有距离，怎么消除？
----------------------------------------
1.几个方法
(1)消除默认边距（margin）
浏览器默认给元素添加margin：

* {
  margin: 0;
  padding: 0;
  box-sizing: border-box;
}
/* 或单独设置div */
.color-block {
  margin: 0;
}

(2)解决inline-block元素空白间隙
当使用display: inline-block时，元素间的换行符会被解析为空格：

测试有效！

.container {
  font-size: 0; /* 父容器消除空白 */
}
.color-block {
  display: inline-block;
  font-size: 16px; /* 单独恢复字体大小 */
  margin: 0;
}

(3)使用Flexbox布局（推荐）
Flex布局会自动消除子元素间隙：

.container {
  display: flex;
}
.color-block {
  flex: 1; /* 等分宽度 */
}


(4)使用Grid布局
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3列等宽 */
  gap: 0; /* 确保间距为0 */
}

(5)浮动元素清除间隙

.color-block {
  float: left;
  margin: 0;
}
.container::after {
  content: "";
  display: table;
  clear: both;
}

(6)负边距抵消间距
.container {
  margin-left: -4px; /* 根据实际间隙调整 */
}
.color-block {
  display: inline-block;
  margin-left: 0;
}


关键点总结：
‌- 消除默认边距‌：全局重置 margin/padding
‌- 布局选择‌：
	✅ Flexbox：现代布局首选方案
	✅ Grid：适合复杂网格布局
	⚠️ inline-block：需处理空白间隙
	⚠️ float：需要清除浮动
- ‌特殊处理‌：
	font-size:0 + 恢复字体大小
	负边距抵消间隙
根据实际需求选择解决方案，Flexbox方案是目前最简洁高效的实现方式。

垂直滚动条设置：隐藏，更细，颜色
----------------------------------------
/* 隐藏滚动条但保留功能（WebKit内核浏览器） */
.main .left::-webkit-scrollbar {
	display: none;
}
.main .left {
	-ms-overflow-style: none;  /* IE/Edge */
	scrollbar-width: none;  /* Firefox */
	overflow-y: scroll;  /* 必须保留滚动功能 */
}



.main .left{
	position: sticky;
    top: 40px;

	max-height: calc(100vh - 95px); /* 视口高度减去顶部偏移，减去底部隐藏目录 */
	overflow-y: auto; /* 启用垂直滚动 */
	padding-bottom: 20px; /*左侧菜单 底部留白*/

	/* 设置滚动条粗细、颜色 */
	scrollbar-width: thin; /* auto | thin | none */
    scrollbar-color: #e0e0e0 #ffffff; /* 滑块颜色 轨道颜色 */
}
