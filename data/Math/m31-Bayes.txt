贝叶斯算法
----------------------------------------
1.传通频率派无法解决的
- 小概率事件需要很多实验
- 有些不好实验：比如日本地震的概率
- 给出的不是一个频率，而是一个概率分布


(2)解决方法：
- 现有一个概率
- 新增概率时，不断更新后验概率





2.基本概念

(1) 分布函数 Distribution Function
随机变量小于某个值的函数，即F(x)=P(X < x);
等同于 累积密度函数(Cumulative Density Function)

- 对于连续型分布，分布函数或累计密度函数 是 概率密度函数(Probability Density Function)的积分:
	F(x) = 积分(-无穷大, +无穷大,  f(x)dx )

- 对离散型分布来说，分布函数或累计密度函数 是一个阶梯状的分段函数。


(2) 概率密度函数 Probability Density Function

三大定律
----------------------------------------
1.条件概率
AB同时发生的概率记作 P(A∩B)，或 P(AB)。

(1)条件概率定义 P(A|B)=P(AB)/P(B)
所以 P(AB)=P(A|B)*P(B)

(2)同理 P(AB)=P(B|A)*P(A)=P(A|B)*P(B)
P(A|B)=P(A)*P(B|A)/P(B)

(2)如果A和B互相独立，则P(A∩B)=P(A)*P(B)
此时: P(A∩B)=P(A)*P(B|A)=P(A)*P(B)
P(B|A)=P(A)*P(B)/P(A)=P(B)





2.全概率公式
若n个事件两两互斥，且这n个事件的并是Ω，则称这n个事件为完备事件组。
符合 Y={y1, y2, ...}，且 P(Y)=1

则 P(X)=P(X,Y)=P(X,y1) + P(X,y2) +...
	=P(X|y1)*P(y1) + P(X|y2)*P(y2) +... 

(1)如果Y只有2种情况
P(X)=P(X|Y)*P(Y) + P(X|~Y)*P(~Y)





3. 贝叶斯定律
类比垃圾邮件识别案例，
X是一个具体的邮件
C是条件：是否垃圾邮件。

P(C|X)=P(X|C)*P(C)/P(X)=P(X|C)*P(C)/累加( P(X|Ci)*P(Ci) )

其中，
	P(C) 是先验概率
	P(C|X) 是后验概率，X已经发生的情况下，C发生的概率。

(1) 贝叶斯分类器
就是比较P(c1|X)/P(c2|X) 比值和1的关系，根据X判定其最可能的分类。

C_hat = argmax(c)P(C|X1,X2,...,Xd)

(2) 实例
上课男女比例是1:2。
其中男生中长发1/30，拖鞋4/5；
女生中长发9/10，拖鞋1/10。
现在有一个X1(长发、拖鞋)，判断性别？
另一个X2(短发、无拖鞋)，判断性别？

解答:
C={M,F}
	P(C=M)=1/3
	P(C=F)=2/3
X={条件}
	P(长发|C=M)=1/30
	P(拖鞋|C=M)=4/5
	
	P(长发|C=F)=9/10
	P(拖鞋|C=F)=1/8

P(C|X)=P(X|C)*P(C)

Ratio P(C=M|X)/P(C=F|X) = P(X|C=M)*P(C=M) / P(X|C=F)*P(C=F)

1) Ratio1 = (1/30*4/5*1/3) / (9/10*1/8*2/3)=0.1 <1, F
2) Ratio1 = (29/30*1/5*1/3) / (1/10*7/8*2/3)=1.1 >1, M
